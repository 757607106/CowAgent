# CowAgent Web 控制台 — 页面功能清单

> 用途：供 Figma 设计前端页面的排版与功能交互。仅列出核心功能，基于实际源码。
> 项目定位：CowAgent 是一个多租户 AI Agent 中台。平台超管管理模型和租户生命周期，每个租户拥有独立的 AI 员工、技能、MCP 连接、渠道绑定、知识库、用量和成员体系，租户与租户之间的数据完全隔离。
> 前端Chat基于Ant Design x




---

## 1. 全局框架

### 1.1 双角色菜单体系

系统存在两种角色，菜单完全不同：

**租户用户菜单：**

| 菜单 | 路由 |
|---|---|
| 对话 | `/chat` |
| 租户模型 | `/tenant-models` |
| AI 员工 | `/agents` |
| 技能 | `/skills` |
| MCP | `/mcp` |
| 渠道接入 | `/channels` |
| 用量 | `/usage` |
| 租户成员 | `/tenant-users` |
| 任务 | `/tasks` |
| 日志 | `/logs` |

**平台超管菜单：**

| 菜单 | 路由 |
|---|---|
| 平台模型 | `/platform/models` |
| 租户管理 | `/platform/tenants` |

### 1.2 应用壳层

- 桌面端：左侧固定 Sider（220px），顶部品牌 `CowAgent 2.0.6`，下方菜单。
- 移动端：顶部 Header + 汉堡菜单按钮，点击打开 Drawer（280px）复用桌面菜单。
- 登录后侧边栏显示：角色 Tag（平台超管/租户名）、用户名、退出登录按钮。

### 1.3 认证

启动调用 `/api/auth/check`，三种状态：

- **检查中**：全屏 Spin。
- **已认证/无需认证**：进入控制台。
- **未认证**：进入登录页。

**登录页（Tab 切换）：**

| Tab | 显示条件 | 字段 |
|---|---|---|
| 平台初始化 | `platform_bootstrap_required` | 平台账号、姓名、密码、确认密码 |
| 登录 | 始终显示 | 账号（租户模式）、密码 |
| 注册租户 | 租户认证模式 | 团队名称、登录账号、姓名、密码、确认密码 |

- 密码最少 8 位，校验两次一致。
- 注册/平台初始化成功后自动登录。
- 退出登录调用 `/api/auth/logout`。

### 1.4 全局运行上下文

- `tenantId`：来自登录用户。
- `agentId`：持久化到 localStorage，通过 Agent 下拉切换。
- `agentOptions`：通过 `listAgentsSimple` 加载下拉选项。
- 切换 Agent 时清空 binding。

## 2. 对话页 `/chat`

两栏布局：左侧会话栏 + 右侧聊天区。

### 2.1 会话栏

- 加载最近 200 条会话，按时间分组（今天/昨天/更早）。
- 每条会话显示：标题、时间、消息数。
- 操作：新建会话、刷新、切换、删除（带确认）。
- 当前会话高亮，会话 ID 按作用域持久化到 localStorage。
- 空态：Empty + 新建按钮。

### 2.2 聊天消息区

**消息角色：**

- **User**：Markdown 文本 + 附件列表（图片缩略图/文件名标签）。
- **Assistant**：ThoughtChain 步骤流 + 主回答 Markdown + 引用来源 + 媒体。
- **Divider**：上下文清理分割线。

**Assistant 消息步骤流：**

- 深度思考步骤：可折叠，显示耗时。
- 工具调用步骤：工具名 + Input/Output 面板，支持折叠/复制，状态（执行中/成功/失败/中止）。
- 阶段进展/阶段回答。
- 主回答：Markdown（代码高亮 + Mermaid 图表 + 视频播放器）。
- 引用来源：折叠列表，最多 8 个 HTTP 链接。
- 媒体附件：image/video/file。
- 成功回复底部提供复制按钮。

**流式状态：**

- 初始"生成中…"占位。
- SSE 事件：reasoning / tool_start / tool_end / phase / delta / text / message_end / image / video / file / cancelled / error / done。
- 用户可中止回复（停止按钮）。
- 历史消息最多展示 200 条，超出显示提示。

### 2.3 输入区

- 多行输入框，autoSize 2~6 行。
- 发送按钮 / 流式中显示停止按钮。
- 输入 `/` 触发 Slash 命令建议浮层（21 条命令）。
- 上下方向键切换最近 50 条输入历史。
- 支持粘贴/拖拽上传文件（拖拽时高亮提示）。
- 附件面板：点击按钮展开，支持多选上传、删除待发附件。
- Footer 工具栏：附件按钮、深度思考开关、Agent 下拉选择器、已选附件数标签。

### 2.4 空态

无消息时显示 Welcome + 3 个快捷 Prompt 卡片，点击直接提交。

## 3. 租户模型页 `/tenant-models`

三个 Tab：可用模型、平台模型、自有模型。

### 3.1 可用模型 Tab

只读表格，展示租户可使用的所有模型（平台+自有）。

列：名称、模型、厂商、来源（平台/本租户）、启用、API Key 状态。

### 3.2 平台模型 Tab

只读表格，仅展示 scope=platform 的模型。

### 3.3 自有模型 Tab

表格额外列：API Base、操作（编辑/删除）。

**新增/编辑弹窗字段：**

- 模型名称（必填）。
- API Base（必填）。
- API Key（编辑时留空保持原值）。
- 启用开关。

---

## 4. AI 员工页 `/agents`

### 4.1 大厅视图

顶部工具栏：卡片/表格切换、租户选择、搜索（名称/ID/模型）、刷新、新员工入职。

概览统计卡片（表格视图时显示）：在岗员工数、可独立服务数、带知识库数、能力连接总数。

**卡片视图每张卡片：**

- 封面区（渐变背景 + 头像 + 状态脉冲）。
- 名称 + 状态 Badge（可服务/基础对话/待配置）。
- 标签组：官方认证、知识库/通用行业、AI助手/基础对话、外部服务/模型。
- 系统提示摘要（2 行）。
- 能力计数 + 工作线说明。
- 操作：配置、应用（跳转对话页）。

**表格视图列：**

员工（头像+名称+状态+ID）、模型、能力（工具/技能/MCP 数量）、知识库、租户、操作（配置/删除）。

默认 Agent 不可删除。

### 4.2 新员工入职弹窗

字段：租户、员工 ID（可选）、名称（必填）、模型（必填，可搜索）、角色设定、工具（多选）、技能（多选）、启用知识库。

### 4.3 Agent 详情页

顶部 Hero：返回大厅、头像+名称+状态、租户/ID、发起会话、保存变更。

**Tab 1 — 核心配置：**

名称、模型（Select）、角色设定（TextArea）、启用知识库开关。

**Tab 2 — 外接能力：**

- 工具能力：Checkbox 列表 + 计数。
- 技能能力：Checkbox 列表 + 计数。
- MCP 连接：Checkbox 启用/禁用列表 + 计数。
- 默认 Agent 工具/技能为空时继承全部。

**Tab 3 — 长期记忆：**

- 子 Tab：记忆与梦境（嵌入 MemoryPanel）。
- 子 Tab：知识库（嵌入 KnowledgePanel）。

---

## 5. 技能页 `/skills`

### 5.1 统计 + 筛选

统计卡片：全部、已启用、系统内置、本地技能。

筛选工具栏：搜索（名称/描述/分类）、状态筛选（全部/已启用/已停用）、来源筛选（全部/系统内置/本地技能）。

排序规则：已启用 > 系统内置 > 名称字母。

### 5.2 卡片视图

每张卡片：图标（认证/代码）、展示名、真实 name、状态 Badge、来源 Tag、分类 Tag、版本 Tag、描述（3 行）。

操作：Switch 启用/禁用、删除按钮（系统内置禁用+Tooltip，本地技能 Popconfirm）。

### 5.3 表格视图

列：技能（图标+展示名+name）、来源、分类、描述、状态、操作（启停/删除）。pageSize=10。

---

## 6. MCP 页 `/mcp`

### 6.1 顶部

Agent 选择器 + 刷新 + 新增服务器。

状态：Agent 加载中/失败（Alert+重试）/无 Agent（Empty）。

### 6.2 Agent 摘要卡片

头像、名称、ID、模型 Tag、工具/技能/MCP Server 数量 Tag。

### 6.3 MCP Server 列表

每个 Server 卡片：名称、command、args、环境变量数。

操作：查看工具/收起工具、测试连接、编辑、删除（Popconfirm）。

**工具展开区：**

- 懒加载工具列表。
- 操作：重新发现。
- 状态：加载中 / 失败 Alert / 空态 / 工具卡列表。
- 工具卡：工具名、描述、点击查看输入 Schema。

### 6.4 新增/编辑弹窗

字段：服务器名称（编辑时禁用）、启动命令、参数（空格分隔）、环境变量列表（变量名+值，可增删）。

Footer：测试连接 + 取消 + 确认保存。

测试结果：成功显示工具卡列表，失败显示 Alert。

### 6.5 Schema 弹窗

标题 `server / tool`、描述 Alert、JSON Schema 代码块。

### 6.6 嵌入模式

嵌入 Agent 详情时：不显示页面标题，按钮文案为"刷新连接/新增连接"。

## 7. 渠道接入页 `/channels`

页面标题：渠道接入。Tab 切换：绑定 / 渠道。

路由 `/channels` 默认进渠道 Tab，`/bindings` 默认进绑定 Tab。

### 7.1 绑定 Tab

工具栏：租户过滤、刷新、新建绑定。

表格列：名称、租户、渠道（Tag）、渠道配置、智能体、启用（是/否）、操作（编辑/删除）。

展开行：JSON 展示 metadata。pageSize=20。

**新建/编辑绑定弹窗字段：**

名称、渠道类型（Select: Web/微信/飞书/钉钉/企微/QQ/公众号/服务号）、渠道配置（关联已有配置）、智能体、启用开关、Metadata(JSON)。

### 7.2 渠道 Tab

工具栏：刷新、新增渠道配置（owner/admin 可见）。

表格列：名称、渠道（Tag）、状态（启用/停用）、回调路径（可复制/扫码登录/长连接）、密钥状态、操作。

操作按渠道类型不同：

- 微信：扫码登录按钮。
- 企微智能机器人：扫码授权按钮。
- 通用：编辑、删除。

**新增/编辑渠道配置弹窗：**

名称、渠道类型（Select，编辑时禁用）、根据渠道类型动态渲染配置字段（text/secret/number/bool/list）、启用开关。

**微信扫码弹窗：**

QRCode 组件渲染二维码 + 状态文字（等待扫码/已扫码请确认/扫码成功/已过期）+ 刷新二维码按钮 + 自动轮询。

## 8. 用量页 `/usage`

筛选栏：Agent 选择器、日期选择器、刷新。

汇总统计（4 列）：租户总 Token、租户总费用、Agent/筛选 Token、MCP 调用数。

明细表格列：时间、Agent、模型、总 Token、输入、输出、工具、MCP、错误、费用、请求 ID。pageSize=20。

---

## 9. 租户管理页 `/tenants`

表格列：名称、状态（active Tag）、操作（编辑）。

展开行：JSON 展示 metadata。pageSize=20。

**新建/编辑弹窗字段：** 名称、状态、Metadata(JSON)。

新建按钮仅非租户认证模式显示。

---

## 10. 租户成员页 `/tenant-users`

工具栏：租户过滤、刷新、新建成员。

表格列：租户、用户 ID、姓名、角色（Tag）、状态（Tag）、操作（编辑/身份映射/删除）。

展开行：JSON 展示 metadata。pageSize=20。

**新建弹窗字段：** 登录账号（可选，自动生成）、姓名、角色（Select）、状态（Select）、Metadata(JSON)、初始密码（填账号时必填，≥8 位）。

**编辑弹窗字段：** 姓名、角色、状态、Metadata(JSON)。

**身份映射 Drawer：**

- 绑定表单：渠道类型（Input）、外部账号标识（Input）、绑定按钮。
- 已绑定列表：渠道、外部账号标识、解绑按钮（Popconfirm）。

---

## 11. 任务页 `/tasks`

表格列：任务标识、名称、状态、下次执行。

展开行：完整 JSON。pageSize=20。操作：刷新。

---

## 12. 日志页 `/logs`

SSE 实时监听 `/api/logs`，最多保留 2000 行。

操作：开始/停止监听、清空日志。

状态 Tag：实时监听 / 已停止。

显示当前行数。

---

## 13. 记忆面板（嵌入 Agent 详情）

分类切换：记忆文件 / 梦境日记（Segmented）。

左侧文件列表表格：文件名、类型（Global/Dream/Daily Tag）、大小、更新时间。支持分页（pageSize=20）。

右侧内容预览：点击文件加载 Markdown 渲染。

---

## 14. 知识库面板（嵌入 Agent 详情）

两个 Tab：文档 / 图谱。

**文档 Tab：**

- 左侧：统计信息（篇数+总大小）、搜索框、目录树（Tree 按目录分组，叶子节点显示标题+大小）。
- 右侧：选中文档的 Markdown 渲染。

**图谱 Tab：**

- 节点/关系计数 + 启用状态。
- 图谱 JSON 展示。
- 刷新图谱按钮。

---

## 15. 平台模型页 `/platform/models`（超管专属）

表格列：名称、模型、厂商、启用、租户可见、API Key 状态、操作（编辑/删除）。

**新增/编辑弹窗字段：** 厂商（Select，编辑时禁用，切换联动模型列表）、模型（Select）、API Key（编辑时留空保持原值）、启用开关、租户可见开关。

---

## 16. 平台租户管理页 `/platform/tenants`（超管专属）

表格列：租户 ID、名称、状态、操作（编辑/删除）。

展开行：JSON 展示 metadata。default 租户不可删除。

**新建/编辑弹窗字段：** 租户 ID（编辑时禁用）、名称、状态（Select: active/disabled/deleted）、Metadata(JSON)。

---

## 17. 系统配置页 `/config`（已废弃路由）

`/config` 路由已重定向至 `/tenant-models`（租户用户）或 `/platform/models`（超管）。

原 ConfigPage 仍存在但不再通过菜单导航进入，其功能包括：

- 模型接入三步：选供应商与模型 → 填凭证（API Base / API Key）→ 自定义模型开关。
- 运行策略：预设（稳健/平衡/高推理）+ 4 个参数（enable_thinking / max_context_tokens / max_context_turns / max_steps）。
- 安全设置（非租户认证模式）：控制台访问密码。
- 底部 sticky 未保存提示条。
